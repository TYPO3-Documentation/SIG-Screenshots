Config.json file¶

This is a json object with the following two keys:

settings¶
Required:false
Type:object

Can be used to override the general settings.

"settings": {
   "snippet": "ignore"
}
comments¶
Required:true
Type:object

These comments are used for the corresponding files in the first line. The comment can be removed by Editors to protect files against being automatically overridden.

{
   "php": "// Automatic screenshot: Remove this comment if you want to manually change this file",
   "rst.txt": ".. Automatic screenshot: Remove this line if you want to manually change this file",
   "general": "Automatic screenshot:"
}
extensions¶
Required:true
Type:array of extension configurations

All actions are stored into the corresponding extension configurations, usually named after the extension (Core or third party) which was is used to create the screenshots.
